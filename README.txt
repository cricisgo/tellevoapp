TeLlevo App es una aplicación pensada para brindarte seguridad de vuelta a tu casa desde Duoc y para acortar los tiempos de viaje de vuelta a tu hogar. ¡Ponte de acuerdo!

Para ejecutar este software, primero deberás descargar NodeJS:
https://nodejs.org/es/

Y deberás instalar algunos paquetes con los siguientes comandos:

npm install -g @angular/cli
npm install –g @ionic/cli
npm install --save @ionic/storage-angular 
npm install --save @ionic/storage
